## 강의 목표

- spring boot & JPA와 친해지기
- 전반적인 흐름 익히기

## 강의 내용

````
1. 스프링 부트 프로젝트 생성 / 빌드 및 실행 해보기
2. 웹 개발 기초
   1. 정적 컨텐츠
   2. MVC와 템플릿 엔진
   3. API
3. 회원 관리 예제로 백엔드 개발
   1. 요구사항 분석 (회원가입 시 이름 중복 불가)
   2. 회원 도메인과 레파지토리 생성
   3. 회원 레파지토리 테스트 케이스 작성
   4. 회원 서비스 개발
   5. 회원 서비스 테스트
4. 스프링 빈과 의존 관계 
5. 회원관리 예제(WEB MVC 개발)
6. DB 접근 기술
7. AOP
````
